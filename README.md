# Movies-Battle
Movie battle. Game based on imdb score

Batalha de filmes do imdb.

![](front-game.png)

http://localhost:4200/

API:
![](swagger.png)
![](swagger-full.png)

Como rodar:
Na pasta do backend execute os comandos
- mvn install
- mvn spring-boot:run

Na pasta do frontend execute os comandos
- npm install
- npm start
